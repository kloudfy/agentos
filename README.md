# AgentOS

Production-ready agent orchestration framework with quality gates.

## Features
- Event-driven architecture
- Plugin system
- Eval harness with quality gates
- Phantom branching (multi-personality agents)
- Auto-generated ADRs

## Built With Kiro
This project uses Kiro's agentic IDE features:
- Spec-driven development
- Agent hooks for automation
- Steering files for consistency
- MCP integrations

## Setup
```bash
npm install
npm test
```
