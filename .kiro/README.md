# Kiro Configuration

This directory contains all Kiro-specific configuration for AgentOS.

## Structure

- `specs/` - Feature specifications that Kiro implements
- `steering/` - Project conventions and architectural guidelines
- `hooks/` - Automation triggers (e.g., test generation, quality gates)
- `mcp/` - Model Context Protocol tool configurations

## DO NOT add to .gitignore

Per Kiroween Hackathon rules, this directory MUST be committed to demonstrate Kiro usage throughout the project.
